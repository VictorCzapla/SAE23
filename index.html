Présentation du projet

Ce site a été réalisé dans le cadre du projet de fin d'année SAé 23 (situation d'apprentissage et d'évaluation) par un groupe de 4 personnes constitué de :

  • Czapla Victor
  • MASSIOT Clément
  • R
  • M

L'objectif était de réaliser un site web dynamique affichant des données provenant de capteurs répartis dans les bâtiments de l'IUT de Blagnac.
Les données sont récupérées depuis un broker MQTT Mosquitto, pour être ensuite stockées dans une base de données MYSQL. Ces données seront ensuite affichées sur ce site en temps réel grâce aux langages informatiques :

  • HTML pour la structure générale du contenu des pages et les formulaires pour les connexions aux pages restraintes
  • CSS pour le style des pages et la structure visuelle du contenu
  • PHP pour traiter et récupérer les informations de connexion et les mesures des capteurs provenant de la base de données
